MKBGv4.0

This framework is built with jQuery and custom CSS. The goal is to provide Browser-Independence. However this may sacrifice speed. JQuery is slower than raw JavaScript. But even JavaScript needs time to load. You can simply avoid this problem by adding values to the CSS. The code is:


body {
padding-top: 49.2813px;
 }


For small screen use the meta tags:


@media only screen and (max-width: 992px){

body {
padding-top: 50vh;
 }

}


This depends on the number of your menu items. Viewport tag is required. You can use this code and modify it. It is tested on the default browser of Windows 7 Ultimate (Internet Explorer 9).

This project could be combined with PHP and AJAX. For example you could build a Real-Time Chat.
